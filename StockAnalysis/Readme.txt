Capstone.py:
    Inputs: 3 stock tickers.
    Inputs: date range that analysis should be conducted

Using Pandas and matplotlib, Capstone.py analyzes stock data and produces it in graphical representation.
    Graphs produced: 
        Price of the 3 stock tickers throughout the date range.
        Volume of the stocks being traded.
        Total number of stocks that were traded
        Daily returns on the stocks
            This helps answer the questions of how related the three companies are.
        Cumulative returns of the stock
            Cumulative return is computed relative to the day investment is made. If cumulative return is above one, you are making profits else you are in loss.
        Rolling Mean Analysis on 50 and 175 days periods for open stock prices
